## How to Run Project:

- Run ```cp .env.example .env```
- Run ```composer i```
- make a Database named: LaraSume and Import SQL File into it
- Run ```php artisan migrate --seed```
- Run ```npm install```
- Run ```npm run prod``` or ```npm run watch```
